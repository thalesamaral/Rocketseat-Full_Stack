# Layouts e evolução

Abordo a evolução dos layouts em CSS, desde o normal flow até a chegada do flexbox e grid. Exploro a transição do uso de tabelas para o conceito de tableless, e como o flexbox permite alinhamentos variáveis e tamanhos flexíveis. O grid, por sua vez, possibilita trabalhar com colunas e linhas simultaneamente. Essas novas propriedades oferecem mais flexibilidade e facilitam o trabalho com CSS moderno.
